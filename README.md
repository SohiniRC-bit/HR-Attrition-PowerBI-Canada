# 🇨🇦 Canadian HR Attrition Intelligence Dashboard

## Problem Solved
Employee attrition is Canada's #1 HR challenge. This dashboard identifies 
who is leaving, why, and what it costs — enabling data-driven retention decisions.

## Tools Used
- Power BI Desktop
- DAX (CALCULATE, FILTER, DIVIDE, SUMX, VAR/RETURN)
- Excel (data source)

## Key Metrics
- Attrition Rate: 10%
- Total Replacement Cost: $200K+
- Flight Risk Employees identified: 10

## Dashboard Pages
1. Executive Summary — KPIs, attrition by department & province
2. Deep Dive — Scatter analysis, tenure comparison
3. Cost Impact — Replacement cost by department
